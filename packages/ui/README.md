# Rolytics - UI
This is the frontend of the [Rolytics Project](https://rolytics.bot.nu).

## Developement
### Setup
No setup needed yet.

### Startup
Running just the devlopment UI can be accomplished like so:
```bash
pnpm dev:ui
```

## Production
### Setup
No setup needed yet.

### Startup
To run just the production UI you can run:
